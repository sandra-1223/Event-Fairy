About Us

HELLO! WELCOME TO EVENT FAIRY

We are providing various services for six different events. The events are Wedding, Engagement, Bride To Be, Naming Ceremony, Baby Shower and Birthday. The Event Fairy website provides user to select the event and choose all the services for the event. We allows user to sign up and user can register for any of the party event from anywhere. This website serves as a single platform for many events. There will be two types of searches available for the user one is location based and the other is search manually.
we will be conducting games on our website such as quiz, spin the wheel, lucky draw etc. and based on the points, the customers can claim their gift from their favourite restaurant. The next specialty of our website is that we provide both customers and restaurants to give away food for the needy under the option ' food for all '. Under event management, we take up all the events like a bride to be, birthday party, wedding, naming ceremony, save the date, engagement, instant photography, flower decorations, balloon decorations, theme-based dresses, cakes, pastries and catering.
